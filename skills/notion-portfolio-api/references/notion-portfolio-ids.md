# Notion Portfolio IDs — Example Template

> Replace all `YOUR_*` placeholders with your actual Notion page/database IDs.
> Find IDs by opening a Notion page and copying from the URL: notion.so/YOUR_ID

## Portfolio Structure

Portfolio Root
  ID: YOUR_PORTFOLIO_HOME_DB_ID
  │
  ├── Portfolio Hub (page)
  │     ID: YOUR_PORTFOLIO_HOME_PAGE_ID
  │
  ├── Domains
  │     ID: YOUR_DOMAINS_SECTION_PAGE_ID
  │           DB ID: YOUR_DOMAINS_DB_ID
  │
  ├── Projects
  │     ID: YOUR_PROJECTS_SECTION_PAGE_ID
  │           DB ID: YOUR_PROJECTS_DB_ID
  │                 Example project: YOUR_SAMPLE_PROJECT_PAGE_ID
  │
  └── Case Studies
        ID: YOUR_CASE_STUDIES_SECTION_PAGE_ID
              DB ID: YOUR_CASE_STUDIES_DB_ID
                    Example case study: YOUR_SAMPLE_CASE_STUDY_PAGE_ID

## Portfolio Hub Page
- ID: YOUR_PORTFOLIO_HOME_PAGE_ID

## Research/Notes DB
- DB ID: YOUR_RESEARCH_NOTES_DB_ID
